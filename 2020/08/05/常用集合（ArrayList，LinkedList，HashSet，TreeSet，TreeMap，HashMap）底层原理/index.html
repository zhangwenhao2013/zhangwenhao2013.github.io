集合关系

下面按Java线路,分别查看下常用List,Set,Map底层存储结构.

List

ArrayList

ArrayList 底层就如其命名一样是一个数组实现存储;

数组形式的存储结构便于查找; 只需要根据脚表(index), 数据类型的存储长度h;

Obj(index) = 头指针+ index * h; 就可以定位到要查找的数据;

ArrayList 优缺点

有点:

如果不是在数组最尾部实现插入数据,每次数组都要通过native方法arraycopy来实现数据移动;
